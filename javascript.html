⚡ JavaScript Notes

1. Hoisting

Hoisting means JavaScript moves variable declarations to the top of their scope. Only declarations are hoisted, not values. 


console.log(a);
var a = 10;

// Output: undefined
  • Only declarations move upward
  • var gets initialized with undefined
  • Access before assignment is allowed

2. Temporal Dead Zone (TDZ)

TDZ is the time when a variable exists but cannot be accessed before initialization. Applies to let and const. 


console.log(y);
let y = 10;

// Output: ReferenceError
  • TDZ happens with let/const
  • Access before declaration causes ReferenceError

3. Block Scope

Variables declared with let and const exist only inside { } blocks. var ignores block scope. 


{
  let b = 2;
}
console.log(b);

// Output: ReferenceError
  • let/const are block-scoped
  • var is function-scoped

4. switch Statement & Fall-through

switch is used when matching one variable against many fixed values. If break is missing, execution continues (fall-through). 


let plan = "basic";

switch(plan) {
  case "basic":
    console.log("Basic features");
  case "pro":
    console.log("Pro features");
}
  • Use switch for fixed values
  • Missing break causes fall-through

5. Ternary Operator

Ternary operator is a short form of if–else used for simple decisions. 


let cartTotal = 1200;

let msg = cartTotal >= 999
  ? "Free Delivery"
  : "Delivery Charges Apply";
  • Best for two-way decisions
  • Avoid nesting ternaries

6. Execution Context

Execution Context is the environment JavaScript creates before running code. It happens in two phases: Memory Creation and Code Execution. 


console.log(a);
var a = 5;

// Memory Phase: a → undefined
// Execution Phase: a = 5
  • Memory allocated before execution
  • Explains hoisting & scope

7. Array Methods

JavaScript provides built-in array methods like push, pop, shift, unshift, splice. 


let arr = [10, 20, 30];

arr.push(40);
arr.pop();
arr.unshift(5);
arr.splice(1, 0, 99);
  • push → add at end
  • pop → remove from end
  • splice → insert/remove/replace

8. DOM Manipulation

DOM selection allows JavaScript to change HTML elements dynamically. 


let div = document.getElementById("profile");
let p = document.createElement("p");

p.textContent = "Name: Amit";

div.appendChild(p);
  • getElementById selects an element
  • createElement makes new HTML tags
  • appendChild inserts into page
